# WordDeck

**英文词库分类助手** — 一款轻量级的个人英语单词管理工具

## 项目简介

WordDeck 是一个基于 Vue3 + Vite 构建的纯前端单词管理应用，专为个人英语学习设计。它帮助你将零散的英语单词按分类整理归档，方便日常查阅、复习和记忆。

**核心价值：**
- 将你学习过程中遇到的生词集中管理，不再散落各处
- 按分类（如"日常用语"、"科技术语"、"商务英语"等）组织词汇，便于针对性复习
- 支持批量导入，快速将已有词库迁移到 WordDeck
- 纯本地运行，数据存储在浏览器中，无需注册账号，隐私安全

## 功能特性

### 批量导入
- 支持拖拽或点击上传 JSON 文件
- 导入前自动校验数据格式，显示预览列表
- 详细的错误提示（缺失字段、格式错误等）
- 内置示例数据，一键导入快速体验
- 自动跳过重复单词，避免数据冗余

### 分类管理
- 按分类查看单词，快速定位目标词汇
- 显示每个分类下的单词数量
- 支持编辑分类名称，批量更新关联单词
- 全局搜索，支持按单词、释义、分类模糊匹配

### 单词列表
- 表格形式展示单词、释义、分类
- 显示单词添加时间（今天、昨天、X天前）
- 单词删除操作，带确认弹窗防止误删

### 本地存储
- 数据自动保存到浏览器本地存储（LocalStorage）
- 无需后端服务，打开即用
- 数据版本管理，确保界面实时同步

## 快速开始

### 环境要求
- Node.js >= 16.x
- npm >= 8.x

### 安装与运行

```bash
# 1. 克隆项目（或直接使用本地目录）
cd word.categorize

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

启动后在浏览器访问 `http://localhost:3000`（端口被占用时自动切换为 3001）。

### 生产构建

```bash
# 构建生产包
npm run build

# 本地预览构建结果
npm run preview
```

## 数据导入格式

WordDeck 支持通过 JSON 文件批量导入单词。文件格式要求如下：

```json
[
  {
    "word": "example",
    "meaning": "例子，实例",
    "category": "日常用语"
  },
  {
    "word": "computer",
    "meaning": "计算机",
    "category": "科技术语"
  },
  {
    "word": "negotiate",
    "meaning": "谈判，协商",
    "category": "商务英语"
  }
]
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| word | string | 是 | 英文单词 |
| meaning | string | 是 | 中文释义 |
| category | string | 是 | 所属分类 |

### 导入限制
- 文件格式：仅支持 `.json` 文件
- 文件大小：不超过 10MB
- 数据结构：最外层必须是数组 `[]`
- 重复处理：已存在的单词会自动跳过

## 使用场景

1. **日常学习**：阅读英文文章时遇到生词，随时添加到 WordDeck 并归类
2. **备考复习**：将四六级、雅思、托福等考试词汇按类别导入，分类复习
3. **专业词汇**：整理编程、医学、法律等专业领域术语
4. **个人词库迁移**：将其他工具导出的词汇整理成 JSON 格式导入

## 注意事项

- **数据存储**：所有数据保存在浏览器本地存储中，更换浏览器或清除浏览器数据会导致数据丢失
- **数据备份**：建议定期将词库导出为 JSON 文件备份（后续版本将支持导出功能）
- **浏览器兼容**：推荐使用 Chrome、Edge、Firefox 等现代浏览器
